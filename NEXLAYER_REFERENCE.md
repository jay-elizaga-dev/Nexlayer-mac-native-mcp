# Nexlayer Platform Reference

> Source: https://nexlayer.com/llms.txt (fetched 2026-04-27)

## Overview

Nexlayer is an agent-native cloud platform enabling AI coding agents to deploy full-stack applications to production-grade Kubernetes without configuration overhead. The platform operates through a single MCP connection and supports any containerized stack.

## Core Value Proposition

1. **AI-Driven Deployment** — Coding agents analyze projects, generate configurations, build containers, and deploy to production autonomously
2. **Infrastructure Simplicity** — "No config, no DevOps, no friction" approach eliminates traditional deployment complexity
3. **Stack Flexibility** — Supports any containerized application, from Next.js to Django to legacy systems
4. **Production-Grade Defaults** — Auto-scaling, load balancing, zero-downtime deploys, and self-healing included by default

## Pricing Model

Credits-based system at $10 per 1,000 credits. A unified pool covers compute, storage, and egress. Free tier provides 500 credits monthly supporting approximately 50 active users. Annual prepayment offers full-year credits upfront with bonus capacity for seasonal workload spikes.

## Use Cases

- **Full-Stack Hosting** — Complete applications with databases deployed in minutes via agent instruction
- **Backend Services** — Persistent containers eliminating serverless cold starts and timeout constraints
- **Side Projects** — Real infrastructure without limitations, scaling smoothly from free tier to paid tiers
- **Freelance Work** — Multi-stack support enabling single platform deployment across diverse client projects
- **Research & Data Science** — Cloud-scale compute and persistent storage for ML training, data processing, and JupyterLab hosting
- **Open Source Self-Hosting** — Tools like n8n, Metabase, Ghost, NocoDB, Supabase, Appwrite, and Convex with persistent data

## Templates & Guides
